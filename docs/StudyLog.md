# 📖 사전 학습 📖

## 0. 클래스와 객체

<br>

### S-0-1. 클래스와 객체의 개념
  
  - `클래스`
    - 객체를 생성하기 위한 틀
    - ES6이후 도입
    - constructor() 메소드를 포함하여 객체가 생성될때 필요한 초기 상태를 설정한다
    - 클래스 안에는 객체의 행동을 정의하는 메소드들이 들어있다
  - `객체`
    - 클래스의 인스턴스
    - 클래스를 기반으로 생성된 실체
    - new 키워드를 사용하여 클래스의 인스턴스를 생성한다
    - 해당 클래스에 정의된 속성과 메소드를 사용할 수 있다
